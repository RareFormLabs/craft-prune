# Craft Prune

A simple Composer package for the Craft CMS ecosystem.

## Installation

You can install the package via composer:

```bash
composer require rareform/craft-prune
```

## Usage

```php
use rareform\Prune\Prune;

$prune = new Prune();
echo $prune->hello(); // "Hello from Craft Prune!"
```

## Testing

```bash
composer test
```
